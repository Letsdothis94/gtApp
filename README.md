# GT APP
## Getting people together
### GTAPP is a social networking app that allows users to connect with like-minded individuals and join groups that share similar interests and hobbies.

1. React js
2. Ruby on Rails
3. WebSockets
4. Google Maps API
5. JWT TOKENS
6. MUI (Material UI)
